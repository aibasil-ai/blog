# Next.js Blog

This project is a Next.js + TypeScript + Tailwind CSS blog with MDX content.

## Scripts

- `yarn dev` - run development server
- `yarn build` - build for production
- `yarn start` - start production server
- `yarn lint` - run Next.js lint

## Content

MDX posts live in `src/content/posts` and are rendered via the `/posts/[slug]` route.
